# tp_doorlocks
TP Doorlocks, a free script supporting all the frameworks. 

This script was firstly made for TPZ-Framework but we also provide it for RedM Community.

# Installation

When opening the zip file, open tp_doorlocks-main directory folder and inside there will be another directory folder which is called as tp_doorlocks, this directory folder is the one that should be exported to your resources (The folder which contains fxmanifest.lua).
